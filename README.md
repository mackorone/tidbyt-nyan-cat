# tidbyt-nyan-cat
A Nyan Cat app for [Tidbyt](https://tidbyt.com/).

## Quick start
1. Install [`pixlet`](https://github.com/tidbyt/pixlet)
1. Run `pixlet serve nyan_cat.star`
1. Go to [http://localhost:8080](http://localhost:8080)
